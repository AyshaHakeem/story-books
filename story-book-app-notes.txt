Npm packages:
express mongoose connect-mongo express-session express-handlebars dotenv method-override moment morgan passport passport-google-oauth20

connect-mongo - store sessions to the database so that when you reset the server, we dont get logged out
morgan - for  logging in, separate for different mode(dev/prod)
express-handlebars - template engine | layouts are in ./views/layouts
dotenv - config 
method-override: allows put and delete methods in place of put and 
moment- format dates

cross-env : global environment variables

edit script tag in package.json to suit production as well as developemnt env:
	"scripts"{
		"start": "cross-env NODE_ENV=production node app"
		"dev": "cross-env NODE_ENV=development nodemon app"
	}
